Vim Tutorial

Posted on 2020-04-18 Edited on 2020-04-19

Basic command for vim.

Moving the Cursor

⬅️ h ⬇️j ⬆️k ➡️l

Text Editing

x : delete the character under the cursor

i : insert before the cursor.

I : insert at the start of the line.

a : append after the cursor.

A : append at the end of the line

:q! : trash all changes.

:wq : save the changes.

w : move the start of a word

e : move the end of a word

$ : move to the end of the line

Delete Command

dw : delete until the start of the next word.

de : delete to the end of the current word.

d$ : delete to the end of the line.

dd : delete a whole line.

Count for a motion

2w : move the cursor two words forward

3e : move the cursor to the end of three words forward.

0 : move to the start of the line.

d2w : delete two words.

5dd : delete 5 lines.

Undo Command

u : undo the last commands

U : fix a whole line

ctrl+R : to undo the undo’s

Put Command

p : put previously deleted text after the cursor. e.g firstly dd, then p;

Replace Command

rx : replace the character at the cursor with x.

ce : to change until the end of a word. e.g worlds => words, put cursor on the character on l

c [number] motion : similar as the delete. motion : w(word), $(end of line)

Cursor Location

ctrl-G : show your location in the file

G : move to the bottom of the file

gg : move to the start of the file

[number] G : go to line [number]

Search Command

/[text] : search for the phrase in the file. To search for the same phrase again, type n. To search for the same phrase in the opposite direction, type N.

?[text] : search for the phrase in the backward direction.

ctrl-O : go back to where you came from .

ctrl-I : goes forward.

Match Parentheses Search

% : find a matching (, [, { , place the cursor on any (, [, {

Substitute Command

:s/old/new : substitute the first occurence of ‘new’ for ‘old’

:s/old/new/g : apply the substitution globally in the line.

:#,#s/old/new/g : change every occurence of a character string between two lines, #,# are the line numbers

:%s/old/new/g : change every occurence in the whole file

:%s/old/new/gc : find every occurence in the whole file, with a prompt whether to substitute or not.

Execute an external command

:![command] : type the command followed by :! ,

Write Files

:w [file_name] : save changes to new file.

Visual Mode

v : press v start Visual selection. Could move the cursor aournd to select text. Then press the : , at the bottom of the screen :'<,'> will appear.

Replace Mode

R : enter the REPLACE mode

Retrieve and merge files

:r FILENAME : insert the contents of a file.

:r !ls retrieve the output of an external command.

Open Command

o : open a line below the cursor

O : open up a line above the cursor

Copy and Paste

y : copy , use visual mode to select text.

p : paste after the cursor

yw : copy a word

Set option

:set ic : ignore case during the search

:set noic : disable ignoring case

:set hlsearch : highlight the searched results

:set nohlsearch : remove the highlighting of matches

/[text]\c : ignore case for just one search command

Getting Help

:help : enter the help window

ctrl-w : jump from one window to another

Completion

ctrl+D : show a list of commands .

<TAB> : complete the command
